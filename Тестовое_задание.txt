Тестовое задание N22:

Сделать самораспаковывающийся экстрактор-zip файлов.

Дано: 

Имеется zip-архив с паролем и разнообразными файлами. В корне архива лежит файл game.exe. Нужно написать экстрактор, который будет извелекать файлы из архива во временную директорию и после экстрактинга запускать файл game.exe. В процессе экстрактинга необходимо показывать окошко, в котором будет выводится лог экстрактинга - какой файл извлекается в текущий момент. Файты должны извлекаться в деректорию %TEMP%\{random_string} - где {random_string} - случайная строка из 8 символов.

Ограничения и условия:
-OS для работы exe: Windows
-Для создания окошка UI должна использоваться библиотека Qt.
-Zip-архив должен храниться в ресурсах скомпилированного exe файла
-Результирующий exe файл должен запускаться standalone без дополнительных dll библиотек, которые отсутсвуют в стандартной установке windows
-Для zip-экстрактинга можно использовать совершенно любую библиотеку или исходный код.
-Пароль для экстрактинга можно взять произвольный и захардкодить внутри кода.
